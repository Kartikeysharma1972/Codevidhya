What this app is

An AI tutor that understands which class the student is in.

Most AI chat tools are built for adults. AI Tutor is built for school students — every answer is grade-calibrated, every test follows CBSE structure, every project idea respects the syllabus. The student logs in, picks a chapter, and the AI behaves like a tutor who already knows the textbook.

🎯

Grade-Adaptive AI

The student's class is locked at signup and travels with every AI call. A Class 3 student gets stories and analogies. A Class 12 student gets JEE/NEET-level depth.

📚

CBSE Curriculum-Locked

Subjects, chapters, and topics come from the official CBSE/NCERT structure — so the AI never wanders off-syllabus or makes up content.

🧪

Real Exam Engine

Mock tests follow CBSE rules — question counts, time limits, difficulty splits, even question types vary per grade group. Not a generic quiz tool.

💡

End-to-End Toolkit

Doubt → explanation → notes → project → mock test → analytics — all four tools live in one signed-in dashboard with chat history per tool.

Prerequisites

What you need to run AI Tutor.

Setup is light. The app runs in any modern browser — no installs for students. Hosting needs are minimal and the entire stack can run on a single web service.

👤 For Students (End Users)

  • 1
    A modern browser — Chrome, Edge, Safari or Firefox (last 2 versions). No app install.
  • 2
    Internet connection — required for AI calls and saving sessions to the cloud.
  • 3
    An email address for signup. Parents can create the account for younger children.
  • 4
    Knowledge of their class (1–12) — picked once at signup, can be changed any time from the profile.
  • 5
    Any device — desktop, laptop, tablet or mobile. The interface is fully responsive.

🏫 For Schools & Parents

  • 1
    One account per child — locked to their CBSE class so every answer matches their syllabus.
  • 2
    Zero installation overhead — works directly from the browser, nothing to set up on lab machines.
  • 3
    Saved history per student — every chat, summary, project and test stays available for revision.
  • 4
    Safe scope — AI is bounded by the CBSE chapter list, so the child never sees off-syllabus content.
  • 5
    Works on shared devices — students log in/out cleanly, sessions are private and grade-locked.

Why does a pencil look bent when dipped in a glass of water?
Great observation! This happens because of refraction. Light travels at different speeds in air and water — when it crosses the boundary, it bends. Your eyes still trace the light back in a straight line, so the pencil looks shifted from where it actually is.
💡 Real-world example: Same reason a swimming pool looks shallower than it really is — and why divers use special glasses to see clearly underwater.

5

Learn / Test

Chat, summarise, generate, attempt. All sessions saved to history.

Why this matters

FOCUS

One Audience, Done Right

ChatGPT serves everyone — programmers, lawyers, marketers, students. AI Tutor serves one user: a CBSE school student. Every feature is built for that single profile.

Curriculum > Cleverness

The AI is bounded by the official CBSE chapter list. It won't go off-syllabus, won't recommend non-grade content, won't confuse a Class 4 student with Class 10 vocabulary.

Real Exam Engine

This isn't a quiz wrapper. The mock test engine implements CBSE-style question variety, time-per-question rules, and post-test analytics — things a generic model has no concept of.

Built-In Memory

Every chat, summary, test attempt and project session is saved per student. The sidebar gives a full history — students return to past work, parents can review.

Zero Prompt Fatigue

Students don't need to write "act as a Class 8 teacher and explain in simple language" — the platform handles that for them on every call, invisibly.

Parent-Safe by Design

The AI is scope-locked to the child's chapters and class — no random internet content, no off-syllabus drift, no need for parents to monitor every chat session.
